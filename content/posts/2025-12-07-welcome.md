## はじめに

技術ブログを開設しました。このブログでは、日々の開発で学んだことや、試したことを記録していきます。

## このブログの技術スタック

このブログは以下の技術で構築されています:

- **静的サイトジェネレーター**: Hugo
- **テーマ**: PaperMod
- **ホスティング**: Cloudflare Pages
- **CI/CD**: GitHub Actions

## なぜHugoを選んだのか

1. **圧倒的なビルド速度** - Go言語で書かれており、数千ページでも数秒でビルド完了
2. **シンプルな記法** - Markdownで記事を書くだけ
3. **豊富なテーマ** - PaperModをはじめ、多数の美しいテーマが利用可能

## コードブロックの例

このブログではシンタックスハイライト機能が使えます。行番号とコピーボタンも表示されます。

### Go言語の例

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
```

### Pythonの例

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

# フィボナッチ数列の最初の10項を表示
for i in range(10):
    print(f"fib({i}) = {fibonacci(i)}")
```

### シェルスクリプトの例

```bash
#!/bin/bash

# Hugo サーバーを起動
hugo server -D

# ビルド
hugo --minify
```

## 画像の挿入方法

画像は `static/images/` ディレクトリに配置して、以下のように記述します:

