# 技術ブログを作成します。

[前提条件]
- まずはユーザーに設計プランをみせて
- my-blog/ 直下に作成予定です。
- 独自メインは用意してます(Xserver)。
- 技術スタックは
- Hugo(Astroでも可能) GithubActions
    - ローカルでマークダウンを編集して記事を各
    - ただしコードを差し込む部分に関してはストレスなくしたい
    - 爆速サイトにしたい
    - push したらActionsでそのままデプロイされるようにしたい
    - 環境変数・秘匿情報系にはセキュリティを高めて欲しい
    - 頻繁ではないが記事内に画像も差し込めるようにしたい
- 他に便利なツール・技術スタックがあれば提案してもいい
- 可能な限り無料で運用したい
